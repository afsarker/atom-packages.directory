Source Preview

source-preview package

Source Preview for Atom.

Features

  • Live updating of preview
  • Shows error messages
  • Synchronize cursor

Supported Languages

  • CoffeeScript
  • Babel: source-preview-babel
  • React(JSX): source-preview-react
  • React(CJSX): source-preview-react
  • Jade: source-preview-jade

Commands

  • source-preview:toggle

Settings

  • enableBuiltinProvider (default: true)

Keymap

No keymap by default.

edit ~/.atom/keymap.cson

coffeescript 'atom-text-editor[data-grammar~="source"]': 'ctrl-M': 'source-preview:toggle'
