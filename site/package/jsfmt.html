Jsfmt

atom-jsfmt

Format your JavaScript and JSON using jsfmt.

Installation

bash $ apm install jsfmt

Usage

  • Press ctrl + alt + f
  • Packages > jsfmt > Format JavaScript
  • Right Click > Format JavaScript

Selected text will be formatted if there is selection.
