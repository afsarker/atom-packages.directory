Gmod Luadev

GMod LuaDev

Luadev integration for Atom.

Send code to LuaDev with simple shotcuts!

Commands

  • gmod-luadev:sv
    • Send code to the server
  • gmod-luadev:sh
    • Send code to the server and all clients
  • gmod-luadev:cl
    • Send code to all clients
  • gmod-luadev:self
    • Send code to just your client
