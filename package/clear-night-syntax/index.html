Clear Night Syntax by Northern Spiral

Install with: 
    apm install clear-night-syntax

Syntax theme for Atom inspired by the soothing colors of the night sky. Designed for use with Clear Night UI.

Clear Night Syntax is based on Solarized Dark Syntax

Preview

Icons provided by File Icons, shown with Clear Night UI and Minimap.

Getting started

### Installation <img src=”https://atom.io/favicon.ico” alt=”” width=16 height=16 /> apm
shell apm install clear-night-syntax

<img src=”https://atom.io/favicon.ico” alt=”” width=16 height=16 /> Atom
1. Open Settings > Install 2. Set the filter to Themes 3. Search for clear-night-syntax and install

Activation 1. Open Settings > Themes 2. Select Clear Night from the Syntax Theme drop-down menu
