Installation

Atom:

Open Settings > Install Set the filter to Themes Search for icecream-nord-syntax and install

Activation:

Open Settings > Themes Select Icecream Nord from the Syntax Theme drop-down menu

CLI:

apm install icecream-nord-syntax
