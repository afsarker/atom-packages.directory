Installation instructions

Go to Atom > File > Settings then search for Web Audio in the Packages tab. Once found, install. ## Development js $ cd ~/.atom/packages $ git clone https://github.com/njenkins/atom-webaudio-snippets.git $ cd atom-webaudio-snippets $ apm install $ apm link
