atom-fallout-ui

Modern UI with Fallout-ish colors

Atom + Fallout looks like the perfect combination for nerd developers. So I created an Atom dark theme inspired to it!
Feel free to check my syntax theme, which fits very well with this UI: bitcoder-syntax.

(Everything has been made smooth through transitions.)

How to Install

Through command line

shell apm install atom-fallout-ui

Through settings
1. Go to File -> Settings -> Install 2. Click on Themes and type atom-fallout-ui 3. Click on the Install button of the corresponding list item. 4. Activate it in File -> Settings -> Themes from the second drop-down menu.
