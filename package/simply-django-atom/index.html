Simply Django

Django framework extended support for Atom

Description

Simply Django is an Atom editor package made for web developers with a concern of productivity. Its goal is to give an extended support of the Django framework to this text editor. It is continuously updated to be consistent with the latest Django version, actually: 1.10.

Installation

There is three way to install Simply Django: * Open Atom, go to Settings -> Install -> Package Search, and search simply-django-atom * Using Atom Package Manager in your terminal apm install simply-django-atom * In your terminal, go to ~/.atom/packages and execute git clone https://github.com/hectocore/simply-django-atom.git

Early Alpha

Simply Django is actually in an early alpha stage. Please keep in mind that its development has just started.
