material-dark-syntax

A dark material design syntax theme for Atom, with a good contrast, and vibrant pops of colors.

Colors

Install

Atom

  1. Open Settings > Install
  2. Set the filter to Themes
  3. Search for material-dark-syntax and install

Atom Package Manager

shell $ apm install material-dark-syntax

Activate

  1. Open Settings > Themes
  2. Select Material Dark from the Syntax Theme dropdown

About

This color scheme is also available as a shell theme in materialshell.
