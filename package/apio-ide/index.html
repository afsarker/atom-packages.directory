Apio Ide

Install with: 
    apm install apio-ide

Develop hardware for open FPGAs easily

Introduction

Apio-ide is an Integrated development environment based on Atom, Apio and Platformio-ide, for developing hardware for the ICE40 Lattice FPGAs family. These FPGAs are the first that are supported by a fully opensource toolchain: Icestorm project

Generate the bistream from the Verilog files, and then upload it into the FPGA
