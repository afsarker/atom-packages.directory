GruvHub - Syntax Theme for Atom



Install

This repository is distributed with apm. After installing apm, you can install gruvhub-syntax with this command.

$ apm install gruvhub-syntax

Usage

After installation, you can enable this theme in your Atom theme settings.

Supported languages

  • C
  • CSS / Less / SASS / SCSS
  • Python
  • Java

If you need support for specific language, please, open an issue.
