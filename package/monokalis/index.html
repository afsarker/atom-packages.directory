Monokalis

Install with: 
    apm install monokalis

Monokalis syntax-theme

The Monokalis theme is a syntax theme based on the popular Monokai colour scheme. It integrates perfectly with the Seti UI theme.

The background colour is taken from Seti UI. The syntax colours are standard Monokai colours, saturated a bit by blending them with the background colour.

Development

Not all use cases have been covered, as I couldn’t find out what they were used for. They have been marked with the special TODO colour. If you find a use case for any of them, please file an issue.

If there is anything else that you would like to see added/covered, please file an issue as well. I’ll try to edit it as soon as possible.
