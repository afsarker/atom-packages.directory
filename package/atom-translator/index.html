Atom Translator

Translation using Yandex and Google Translation Service

Just select text press ctrl-alt-t and get your text translated into your selected language.

Setup: * Get your Translation API Key for your service provider. * Enter Your Translation key and Translation language(as shortcode) in package setting.

Get API Key At:

  • Yandex Translation api key
  • Google Translation api key
