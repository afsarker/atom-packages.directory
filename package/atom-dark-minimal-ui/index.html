Atom Dark Minimal UI

A dark minimal UI with configurable color.

Installation

Open a terminal and type

shell apm install atom-dark-minimal-ui

or

Via Atom
1. Open File > Settings > Install 2. Type atom-dark-minimal-ui and set the filter to Themes 3. Find and install theme

Activate theme 1. Open File > Settings > Themes 2. Select Atom Dark Minimal from the UI Theme drop-down menu

Configuration

Atom Dark Minimal UI has a configurable color. 1. Open File > Settings > Themes 2. Press the button next to selected UI theme 3. Select a Custom color via the colorpicker or use the Default Color
